DevScope

Site du dépôt GitHub

Le code et les mises à jour du projet sont sur GitHub. Pour ouvrir l’application complète (dashboard, API) dans le navigateur, déployez-la une fois (Vercel + Render) — le dépôt reste la source : chaque push peut redéployer automatiquement.

Application en ligne

Suivez la section Déploiement en ligne du README sur GitHub : API sur Render, interface sur Vercel, variable INTERNAL_API_URL.

Lancer le projet en local

git clone https://github.com/Jacque004/DevScope.git
cd DevScope
npm install
cp .env.example .env
npm run dev

Puis ouvrez http://localhost:3000
