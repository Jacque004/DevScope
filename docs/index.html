DevScope

Page GitHub Pages

Ce site ne contient pas l’application Next.js : GitHub Pages sert uniquement des fichiers statiques. L’interface complete (dashboard, comparaison) et l’API NestJS se lancent sur votre machine ou sur un hebergeur (Vercel, Render, etc.).

Utiliser l’app en ligne (sans cloner)

Suivez la section Déploiement en ligne du README sur GitHub : API sur Render, interface sur Vercel, variable INTERNAL_API_URL.

Lancer le projet en local

git clone https://github.com/Jacque004/DevScope.git
cd DevScope
npm install
cp .env.example .env
npm run dev

Puis ouvrez http://localhost:3000
